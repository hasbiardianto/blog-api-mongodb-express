
# BLOG POST MONGODB + EXPRESS API

Blog post api using mongodb and express.js. noSQL database a collection include title, slug and post.

## Prerequisted

- bcrypt: ^5.1.1
- connect-mongo: ^5.1.0,
- cookie-parser: ^1.4.6,
- dotenv: ^16.4.2
- express: ^4.18.2
- express-session: ^1.18.0
- jsonwebtoken: ^9.0.2
- method-override: ^3.0.0
- mongoose: ^8.1.1
